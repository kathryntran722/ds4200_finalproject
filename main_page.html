Exploring Cancer–Gene Associations

An interactive data visualization project using the Genetic Association Database (GAD) to uncover patterns in how genes, chromosomes, and cancer types have been studied worldwide.

Motivation

About the Data

This project uses the Genetic Association Database (GAD), a curated archive of published human genetic association studies. We focus specifically on the cancer subset of this database.

Dataset size

The dataset contains approximately 5,000+ records, each representing one published study. Studies span from 1999 to 2006 and cover hundreds of genes across all 23 human chromosomes.

Key attributes

Each record includes the following fields:

  • gene — the gene variant being studied (e.g. ABCB1, BRCA1)
  • phenotype — the cancer type being investigated (e.g. breast cancer, leukemia)
  • association — whether the study found a significant link: Y (yes), N (no), or blank (not coded)
  • chromosome / chromosome_band — the genomic location of the gene
  • year — the publication year of the study
  • population / continent — the geographic or ethnic group studied
  • reference / pubmed_id — the source publication

Important note on the association field

The majority of entries have a blank association value — this does not mean no association was found. It means the GAD curators did not code that entry. Only entries explicitly marked Y or N should be interpreted as confirmed or denied associations.
